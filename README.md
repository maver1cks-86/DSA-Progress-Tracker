## How This Works

### Phase Level

* Each **Phase folder** represents a stage of preparation.
* Every phase contains a `plan.md` file.

### `plan.md`

* Outlines:

  * Daily schedule
  * Topics to study
  * Key targets

---

## Daily Structure

Each phase contains folders for **each day (based on date)**:


## Daily Workflow

For each day:

1. Check `plan.md` in the phase
2. Study the assigned topic
3. Watch relevant videos
4. Solve problems
5. Document everything:

   * Add problem files in `Problems-Solved/`
   * Update `videos.md` in `Videos-Watched/`
